# 我们的书库

一个简单好用的本地书籍库存管理桌面应用，帮助你管理书籍的入库、出库、库存、价格和利润。

## 环境要求

- Node.js 18 或以上
- npm 9 或以上
- Windows 或 macOS

## 安装

```bash
# 1. 克隆代码
git clone https://github.com/lijunling77/book.git
cd book

# 2. 安装依赖
npm install

# 3. 编译原生模块（better-sqlite3 需要针对 Electron 编译）
npx electron-rebuild

# 4. 编译主进程
npm run build:main
```

## 运行

```bash
npm run electron:dev
```

会自动打开桌面窗口。首次运行会自动创建数据库。

## 打包

```bash
# 打包成安装包（Windows 生成 .exe，Mac 生成 .dmg）
npm run electron:build
```

打包产物在 `release/` 目录下。

## 功能说明

### 仪表盘

打开应用后的首页，显示：
- 总库存量
- 今日入库数量和金额
- 今日出库数量和金额
- 本月利润
- 最近5条入库记录
- 最近5条出库记录

### 书籍管理

管理所有书籍的基本信息。

**新增书籍：**
1. 点击"新增书籍"
2. 填写书名（必填）、作者、描述（都可选）
3. 点击确定

**编辑书籍：** 在列表中点击"编辑"，修改后确定。

**删除书籍：**
- 没有交易记录的书籍可以直接删除
- 有过入库或出库记录的书籍无法删除（保护利润数据）

**搜索：** 输入关键词，点击"查询"按书名或作者搜索，点击"重置"清空。

**书名唯一：** 不允许创建同名书籍。

### 位置管理

管理存放位置的字典，方便入库时快速选择。

- 新增位置：如"A架3层"、"B柜2格"
- 删除不需要的位置
- 入库时输入新位置会自动加到字典

### 入库管理

记录书籍的采购入库。

**新增入库：**
1. 点击"新增入库"
2. 选择书籍（不存在可点"快速新增"直接创建）
3. 选择或输入存放位置（从字典选择或手动输入）
4. 入库日期默认今天，可修改
5. 填写数量、买入价格
6. 供应商可选填
7. 点击确定

**批量入库：**
1. 点击"批量入库"
2. 顶部"统一日期"可一次设置所有行的日期
3. 每行填一条记录，点"添加一行"增加更多
4. 点击提交

**批量导入入库数据（Excel/CSV）：**
1. 在"数据导入"页面，下载入库导入模板
2. 模板包含示例数据，列：书名、作者、存放位置、入库日期、数量、买入价格、供应商
3. 必填：书名、入库日期、数量、买入价格
4. 日期格式：YYYY-MM-DD（如 2026-01-15）
5. 书籍不存在时自动创建，位置不存在时自动加到字典
6. 上传文件导入，显示结果摘要

**存放位置说明：**
- 位置跟着入库记录走，每次入库可以指定不同位置
- 同一本书可以在多个位置（如入库10本到A架，再入库5本到B架）
- 综合报表会汇总显示该书所有位置

**编辑入库记录：** 可以修改日期、数量、价格、供应商、位置。修改数量会自动调整库存。

**删除入库记录：** 删除后对应库存会自动减少。

### 出库管理

记录书籍的销售出库。

**新增出库：**
1. 点击"新增出库"
2. 选择书籍
3. 出库日期默认今天，可修改
4. 填写数量、售出价格
5. 买家可选填
6. 点击确定

**注意：** 出库数量不能超过当前库存。

**批量出库：** 操作方式同批量入库。

### 库存查询

查看所有书籍的当前库存情况。

- 按书名搜索
- 按状态筛选（正常/缺货）
- 切换"汇总视图"查看每本书的总库存
- 点击列标题可排序
- 显示最近买入/售出价格、平均价格、价格范围

### 价格历史

查看每本书的价格变化。

- 列表显示所有书籍的价格概览（最近价格、平均价格、价格范围）
- 点击行末尾的展开按钮，查看该书的每一笔买入和售出记录明细

### 利润统计

按月或按年查看利润。

- 顶部切换"按月"或"按年"
- 显示每个时间段的入库数量、出库数量、采购成本、销售收入、净利润
- 顶部卡片显示总计

### 综合报表

一张大表看所有信息。

- 每行一本书，包含：书名、作者、位置、库存数量、入库出库汇总、价格信息、利润
- 支持按书名和位置搜索
- 支持按日期范围筛选
- 点击"导出 Excel"或"导出 CSV"导出报表
- 导出文件名格式：`综合报表_2026-04-28.xlsx`

### 库存盘点

核对实际库存和系统记录。

**创建盘点：**
1. 点击"创建盘点任务"
2. 系统加载所有有库存的书籍

**执行盘点：**
1. 点击"查看详情"进入盘点
2. 在"实际数量"列输入你实地清点的数量
3. 系统自动计算差异（盘盈/盘亏）
4. 点击"提交并调整库存"
5. 确认后系统自动调整库存

**盘点调整记录：** 盘盈自动生成入库记录（供应商标记为"盘点调整（盘盈）"），盘亏自动生成出库记录（买家标记为"盘点调整（盘亏）"），确保入库出库数据和库存一致。

### 数据导出

导出数据为 Excel 或 CSV 文件。

1. 选择导出类型（入库记录/出库记录/库存信息/利润统计）
2. 选择导出格式
3. 可选设置筛选条件
4. 点击"导出"，选择保存位置

文件名格式：`入库记录_2026-04-28.xlsx`

### 数据导入

从 Excel 或 CSV 文件批量导入。

**导入书籍：**
1. 下载导入模板（包含书名、作者、描述三列，附示例数据）
2. 填写数据后上传导入

**导入入库数据：**
1. 下载入库导入模板（包含书名、作者、存放位置、入库日期、数量、买入价格、供应商七列，附示例数据）
2. 填写数据后上传导入
3. 书籍不存在时自动创建，位置不存在时自动加到字典

### 备份恢复

**备份：** 点击"创建备份"，选择保存位置。建议定期备份。

**恢复：** 点击"恢复"，选择备份文件。注意：恢复会覆盖当前所有数据。

- **所有列表都支持点击列标题排序**

## 数据存储

所有数据存储在本地 SQLite 数据库中：
- Windows: `C:\Users\{用户名}\AppData\Roaming\book-management-system\books.db`
- Mac: `~/Library/Application Support/book-management-system/books.db`

## 常见问题

**Q: 启动后白屏？**
A: 确保用 `npm run electron:dev` 启动，不要直接在浏览器打开。

**Q: 数据库初始化失败？**
A: 删除数据库文件后重新启动。

**Q: 删除书籍提示"有交易记录"？**
A: 有过入库或出库记录的书籍不能删除，这是为了保护利润数据。

**Q: 盘点后数据对不上？**
A: 盘点确认时会自动生成调整记录（供应商/买家标记为"盘点调整"），入库出库数据是一致的。

**Q: 导入的日期显示为数字？**
A: Excel 日期格式会自动转换。建议在模板中使用文本格式输入日期（YYYY-MM-DD）。

**Q: 同名书籍无法创建？**
A: 书名必须唯一。如有同名书，可在书名后加区分，如"三体（刘慈欣）"。
